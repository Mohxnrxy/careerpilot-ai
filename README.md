# 🚀 CareerPilot AI

CareerPilot AI is an AI-powered career assistant that helps students, fresh graduates, and job seekers improve their career readiness through resume analysis, skill-gap identification, interview preparation, and personalized career guidance.

## 📂 GitHub Repository

https://github.com/Mohxnrxy/careerpilot-ai

---

## 🎯 Problem Statement

Many job seekers struggle to:

* Understand how strong their resume is
* Identify missing skills for their target role
* Match resumes against job descriptions
* Prepare for technical interviews
* Plan their career growth effectively

CareerPilot AI solves these challenges using Google Gemini AI and an interactive web application.

---

## ✨ Features

### 📄 Resume Analysis

* Extracts resume content from PDF files
* Analyzes resume structure and content
* Provides ATS-style evaluation

### 🎯 ATS Score Evaluation

* Calculates resume quality score
* Identifies strengths and weaknesses
* Suggests improvements

### 📊 Skill Gap Analysis

* Compares user skills with target role requirements
* Highlights missing skills
* Provides learning recommendations

### 🧭 Career Roadmap Generator

* Generates personalized career roadmaps
* Suggests technologies and certifications
* Provides growth path recommendations

### 🎤 AI Interview Coach

* Generates role-specific interview questions
* Helps users prepare for interviews
* Supports multiple career paths

### 📋 JD Matcher

* Compares resumes against job descriptions
* Calculates match percentage
* Identifies missing keywords and skills

### 💼 Job Search Assistant

* Helps users discover relevant opportunities
* Aligns recommendations with career goals

### 📑 PDF Career Report

* Generates downloadable career reports
* Summarizes analysis and recommendations

## 🏗️ Architecture

CareerPilot AI consists of multiple specialized AI modules:

1. Resume Analysis Module
2. ATS Scoring Module
3. Skill Gap Analysis Module

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Mohxnrxy/careerpilot-ai.git
cd careerpilot-ai
```

Create a virtual environment:

```bash
python -m venv .venv
```

Activate environment:

```bash
# Windows
.venv\Scripts\activate

# Linux/Mac
source .venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application:

```bash
streamlit run app.py
```
